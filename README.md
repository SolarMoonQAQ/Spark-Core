星火核心是一款基于Minecraft的Lib模组，目标是在MC内打造一个专属的游戏引擎，减少模组作者的部分重复工作。

目前核心主要有以下功能：

- 可在服务端计算姿态的基岩版动画/模型加载器
- 刚体物理集成及线程优化
- Lua脚本集成
- 支持热重载，具有高度可客制化的的拓展包系统

### 使用库
- [Libbulletjme](https://github.com/stephengold/Libbulletjme)
- [KStateMachine](https://github.com/KStateMachine/kstatemachine)
- [JNLua](https://github.com/MightyPirates/OC-JNLua)
